# TC04 - Verify Minimum Search Keyword Length

## Test Case Information
- **Test Case ID:** TC04
- **Title:** Verify system behavior with a minimum-length search keyword
- **Module:** Search
- **Priority:** Medium
- **Type:** Functional / Boundary

## Description
Verify that the system handles a minimum-length search keyword (for example, 1 character) correctly, without returning incorrect results or causing errors.

## Preconditions
- The website is functioning normally.
- The network connection is stable.

## Test Steps
1. Open a web browser (Chrome, Firefox, Edge, etc.).
2. Navigate to: `https://ecommerce-playground.lambdatest.io/`.
3. Wait until the home page is fully loaded.
4. Locate the search input field in the header.
5. Click on the search input field.
6. Enter the minimum-length keyword `a`.
7. Verify the value `a` is displayed correctly in the input field.
8. Click the **Search** button or press **Enter**.
9. Wait for the search results page to load completely.
10. Verify the URL contains the search parameter `search=a`.
11. Verify the product list is displayed.

## Expected Results
- A list of relevant products containing the character `a` is displayed.
