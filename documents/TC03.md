# TC03 - Verify Maximum Search Keyword Length

## Test Case Information
- **Test Case ID:** TC03
- **Title:** Verify system behavior with an overlength search keyword
- **Module:** Search
- **Priority:** Medium
- **Type:** Functional / Boundary

## Description
Verify that the system handles an unusually long search keyword (boundary test) correctly without causing UI or backend errors.

## Preconditions
- The website is functioning normally.
- The network connection is stable.

## Test Steps
1. Open a web browser (Chrome, Firefox, Edge, etc.).
2. Navigate to: `https://ecommerce-playground.lambdatest.io/`.
3. Wait until the home page is fully loaded.
4. Locate the search input field in the header.
5. Click on the search input field.
6. Enter a 250-character string into the search field.
7. Click the **Search** button or press **Enter**.
8. Wait for the search results page to load completely.
9. Verify the message displayed is: `There is no product that matches the search criteria.`

## Expected Results
- No products are displayed.
- The message `There is no product that matches the search criteria.` is displayed.
