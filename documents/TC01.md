# TC01 - Verify Search Functionality with a Valid Keyword

## Test Case Information
- **Test Case ID:** TC01
- **Title:** Verify search functionality with a valid keyword
- **Module:** Search
- **Priority:** High
- **Type:** Functional

## Description
Verify the search functionality when the user enters a valid keyword and uses the Search button to execute the search.

## Preconditions
- The website has products available that match the intended search keyword.

## Test Steps
1. Open a web browser (Chrome, Firefox, Edge, etc.).
2. Navigate to: `https://ecommerce-playground.lambdatest.io/`.
3. Wait until the home page is fully loaded.
4. Locate and click on the search input field in the header.
5. Enter the valid keyword `Macbook` into the search field.
6. Verify the entered value is displayed correctly in the input field.
7. Click the **Search** button (magnifying glass icon).
8. Wait for the search results page to load.
9. Verify the URL contains the parameter `search=Macbook`.
10. Verify the page title is related to the search.
11. Verify that the product list is displayed and includes at least one relevant product.

## Expected Results
- The user is successfully navigated to the search results page.
- The URL contains the correct keyword parameter: `search=Macbook`.
- The product list is displayed correctly based on the keyword.
- The page title shows `Search - Macbook`.
